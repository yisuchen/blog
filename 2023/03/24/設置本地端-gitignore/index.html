設置本地端 gitignore (全域)

發表於 2023-03-24 Disqus：
做過,又忘記了, 就代表該寫篇文章加深印象

前言

各專案在使用 git 上, 通常都有一份 .gitignore 的檔案
忽略一些”不想”, 或是”不需要”被版本控制的檔案

但在團隊協作專案中, 每個人開發或是慣用工具都不同
也因為這些的不同, 會發現有些工具產生了一些奇奇怪怪的檔案
但不想汙染團隊的 .gitignore 又不想每次出現礙眼怎麼辦?
(個人專案的話, 可以不用繼續往下看了, 反正加進 .gitignore 就對了)

設置全域 .gitignore (對本機所有 git 有效)

  1. 先在你喜歡的路徑創建一個 .your_global_ignore_filename
  2. 設置 git 全域忽略
  3. 跟專案內的 .gitignore 一樣, 將想要忽略的路徑編寫在檔案中

灑花! 完成!
