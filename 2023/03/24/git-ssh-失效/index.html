git ssh 失效

發表於2023-03-24|更新於2023-03-24
|評論數:

前言

原本我正在使用 git commit, 正開開心心的準備 push to github 時, 突然噴了以下的 error

其實問題很簡單

就是 ssh key 出問題了, 在 error 說明內也寫得很清楚
Host key verification failed.
包含 ssh_key 檔案
出問題的在第幾行

解法

  1. 刪除錯誤(舊的)的 key
  2. 重新產生一組新的 ssh_key, 記得添加到 Github 帳戶內
    (key 有 2 把, 私key , 公key)

補充筆記

完成後, 在資料夾內會產生 3 個檔案

  1. id_rsa
  2. id_rsa.pub
  3. known_hosts

再將 id_rea.pub 設置到你需要的地方即可
e.g.

  1. gitlab - user setting - ssh keys
  2. gitgub - setting - ssh keys and gpg keys
  3. sourcetree - setting - ssh key
